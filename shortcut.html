IntelliJ IDEA 단축키 15가지
1. Alt + Enter Suggestions for error correction
2. F2 Navigate errors
3. Alt + 1 Open project window
4. Esc Focus back to editor
5. Ctrl + E View recent files
6. Ctrl + B See declaration of symbol
7. Alt + F7 See usage
8. Ctrl + Ctrl Run anything
9. Shift + Ctrl + W / Ctrl + W Expand / Close selection
10. Ctrl + / Comment Shift + Ctrl + / Comment Block
11. Shift + Ctrl Enter Auto complete
12. Ctrl + Alt + L Reformat file
13. Shift + Ctrl + Alt + T Refactor
14. Shift + Ctrl + A Find action
15. Shift + Shift Search
Alt + enter
Alt+Enter 키보드 단축키를 사용하면 무엇이든 수정할 수 있다.


F2
F2를 활용해 다음 오류, 경고, 제안으로 이동할 수 있다.


Alt + 1
Cmd+1(Mac) 또는 Alt+1(Windows)를 사용해 프로젝트 창을 열수있다.


Esc
Esc를 누르면 에디터로 다시 초점을 옮길 수 있다.


Ctrl + E
Cmd+E(Mac) 또는 Ctrl+E(Windows)를 사용해 가장 최근 열었던 파일을 확인할 수 있다.


Ctrl + B
Cmd+B 또는 Ctrl+B(Windows)를 사용해 심볼의 선언으로 이동할 수 있다.


Alt + F7
Alt+F7을 누르면 모든 사용 위치가 표시된다.


Ctrl + Ctrl
Ctrl을 두 번 누르면 Run Anything 창이 열린다


Shift + Ctrl + W / Ctrl + W
Alt+위/아래 방향키(Mac) 또는 Ctrl+Shift+W(Windows)를 사용해 커서 주변의 코드 섹션을 확장하거나 축소할 수 있다.


Ctrl + / Comment Shift + Ctrl + /
Cmd+/(Mac) 또는 Ctrl+/(Windows)를 사용해 간편하게 주석을 추가할 수 있다.


Shift + Ctrl + Enter
Shift+Cmd+Enter(Mac) 또는 Shift+Ctrl+Enter(Windows)로 현재 구문 완성 기능을 사용할 수 있다.


Ctrl + Alt + L
Alt+Cmd+L(Mac) 또는 Ctrl+Alt+L(Windows)를 사용해 현재 파일의 서식을 프로젝트 표준에 맞출 수 있다.


Shift + Ctrl + Alt + T
Ctrl+T(Mac) 또는 Shift+Ctrl+Alt+T(Windows)를 사용해 모든 리팩토링을 표시할 수 있다.


Shift + Ctrl + A
Shift+Cmd+A(Mac)또는 Shift+Ctrl+A(Windows)를 통해 액션 검색을 할 수 있다.


Shift + Shift
Shift를 두 번 누르면 무엇이든 검색 가능한 검색 창이 표시된다.
